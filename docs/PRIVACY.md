# Privacy & PDPL Notes

Rihla targets travelers in Saudi Arabia and the GCC, so we treat the Saudi **Personal Data Protection Law (PDPL)** as our baseline.

## What we collect

| Category | Source | Storage |
|---|---|---|
| Consent flag | Landing page checkbox | `Session.consentGiven` |
| Device hints | User agent / `Accept-Language` | `Session.deviceType`, `browserLanguage`, `timezone` |
| IP (hashed) | Request socket | `Session.ipHash` (SHA-256, no salt — single-use forensics only) |
| Chat content | User messages | `Conversation.content` |
| Behavioral profile | Inferred by `profiler` | `TravelerProfile.*` |
| Itinerary | Generated by Claude | `Itinerary.itineraryJson` |

We do **not** collect: name, email, phone, payment data, raw IP, or precise geolocation.

## Legal basis

- **Consent** for chat content + profiling: explicit checkbox at session creation
- **Legitimate interest** for rate-limiting and abuse prevention: hashed IP only

## User rights (PDPL Articles 4 & 18)

| Right | How we satisfy it |
|---|---|
| Access | `GET /api/sessions/:id/export` returns the full session JSON |
| Erasure | `DELETE /api/sessions/:id` cascades across all related rows |
| Rectification | User can start a new session and re-converse |
| Withdraw consent | Toggling `consentGiven` halts further profiling |

The session ID is the access key. We surface it in the UI (sidebar) and document export/delete in the privacy page.

## Cross-border transfer

- **Claude (Anthropic)** — chat content and profile sent for inference. No persistence on Anthropic's side beyond their stated retention.
- **PostHog** — analytics events. No raw chat content sent; only event names + metadata.
- **Vercel / Railway** — hosting; data at rest in their managed Postgres (Railway region: see deploy config).

We document these in the privacy page and require consent before any send.

## Retention

- `Session` and child rows: indefinite while the user holds the ID. We provide self-service delete.
- Server logs: 30 days (Railway default).
- Redis cache: TTL ~ 24h.
